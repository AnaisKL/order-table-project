1. 직원(Empl) API
기능 (CRUD),HTTP 메서드,URI 경로,설명,요청 데이터 (Request)
조회 (Read),GET,/employees,전체 직원 목록 조회,(없음)
조회 (Read),GET,/employees/{emp_id},특정 직원 상세 정보 조회,(없음)
등록 (Create),POST,/employees,신규 직원 등록,"사번, 이름, 부서, 연락처"
수정 (Update),PUT,/employees/{emp_id},"직원 정보 수정 (부서 이동, 연락처 변경 등)","변경할 부서, 연락처"
삭제 (Delete),DELETE,/employees/{emp_id},퇴사자 등 직원 정보 삭제,(없음)

2. 비품(Supply) API
기능 (CRUD),HTTP 메서드,URI 경로,설명,요청 데이터 (Request)
조회 (Read),GET,/supplies,전체 비품 목록 조회 (카테고리 검색 포함),(없음)
조회 (Read),GET,/supplies/{item_id},특정 비품 상세 정보 조회,(없음)
등록 (Create),POST,/supplies,신규 비품 등록,"비품명, 카테고리, 초기 재고"
수정 (Update),PUT,/supplies/{item_id},기존 비품 정보 및 재고 수정,"비품명, 카테고리, 수정할 재고"
삭제 (Delete),DELETE,/supplies/{item_id},목록에서 사용하지 않는 비품 삭제,(없음)

3. 신청 내역(Order Request) API
기능 (CRUD),HTTP 메서드,URI 경로,설명,요청 데이터 (Request)
조회 (Read),GET,/orders,전체 신청 내역 목록 조회,(없음)
조회 (Read),GET,/orders/{order_id},특정 신청 건 상세 조회,(없음)
등록 (Create),POST,/orders,비품 신규 신청,"사번, 비품 번호, 신청 수량"
수정 (Update),PUT,/orders/{order_id},신청 상태 변경 ('대기중' → '지급 완료' 등),변경할 상태 값
삭제 (Delete),DELETE,/orders/{order_id},잘못된 신청 건 취소 및 삭제,(없음)
